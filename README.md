## Routes

<ul>
<li><b>/</b></li>
"message" : "Hi, this is Random Anime Qoute API"
<li><b>/random-anime-quote</b></li>
"quote" : An anime quote <br/>
"character" : A character who said it <br/>
"anime" : That anime
</ul>
